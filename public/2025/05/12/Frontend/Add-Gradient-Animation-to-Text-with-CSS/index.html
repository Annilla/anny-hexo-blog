Add Gradient Animation to Text with CSS

5月 12, 2025 分類 Frontend

最近工作上需要設計將文字當作圖樣的 Logo, 起因為了減少後續維護成本，不打算針對每個部門製作專屬圖片。取而代之，會在介面中直接使用部門名稱，並搭配漸層動畫，讓每個部門既能有自己的識別感，又不需要額外製圖，省時省力又有彈性。 Demo 放在 Codepen。

CSS

話不多說, 直接上 Code. HTML 直接灌一個 gradient-text class, 其他想要的地方都能貼上套用相同 class 超方便!
