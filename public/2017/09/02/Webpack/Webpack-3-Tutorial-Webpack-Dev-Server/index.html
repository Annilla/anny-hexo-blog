疑？內容的文字怎麼好像被切掉了。我們趕緊來試試看修改 css 的部分，順便看看 Hot Reload 的威力吧～

關於實時加載（Live Reload）和熱加載（Hot Reload）的區別在於:實時加載應用更新時需要刷新當前頁面，可以看到明顯的全局刷新效果;而熱加載基本上看不出刷新的效果，類似於局部刷新。
